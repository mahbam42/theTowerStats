A stats-tracking and analysis app for **The Tower** mobile game.

It imports Battle Report text, preserves raw values, and presents charts and read-only progress dashboards without prescriptive strategy recommendations.

## What the App Includes

- Battle Report import with deduplication and non-fatal handling of unknown labels
- Charts with filters (date range, tier, presets) and exports (CSV, PNG)
- Read-only progress dashboards for cards, ultimate weapons, guardian chips, and bots
- Per-account data isolation and an optional read-only demo dataset

## What the App Does Not Include

- Prescriptive strategy recommendations
- Automatic optimizers
- Real-time wiki scraping
